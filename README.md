# PENSAMENTOS

# RESUMO
<p> Pensamento é um projeto que irá exibir varios pensamentos dentro do App dinamicamente e animada. O objetivo desse projeto, é realizar a pratica do que foi visto no curso do <b>Eric Brito</b> na Udemy.</p>
